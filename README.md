# Road Accident Dashboard (2021–2022)

## Project Overview
This project analyzes road accident data for 2021 and 2022 to generate insights on casualties, accident severity, and contributing factors. The dashboard was built in Microsoft Excel using pivot tables, charts, and data modeling techniques.

## Objectives
- Analyze total casualties from road accidents
- Compare accident severity and casualty percentages
- Identify trends by vehicle type
- Track monthly casualty trends (2021 vs 2022)
- Analyze impact of road surface conditions
- Understand relationship between location and time (Day/Night)

## Key KPIs
- Total Casualties
- Casualties by Severity (Fatal, Serious, Slight)
- Casualties by Vehicle Type
- Monthly Trends (YoY comparison)
- Casualties by Road Surface
- Casualties by Area and Time (Day/Night)
